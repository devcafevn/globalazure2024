Agenda (Vietnamese)

08:20 AM - 08:30 AM
Opening
8:30 AM - 09:15 AM

Microsoft Entra ID for developers

Thien Nguyen

Learn basic concepts of Microsoft Entra ID (formerly known as Azure Active Directory). How to manage role-based access control in Azure and how to implement Single Sign-On (SSO) for your applications with Entra ID.

09:15 AM - 10:00 AM

Seamlessly Integrating Azure Kubernetes Service with Azure KeyVault Using Workload Identity

Phuc Nguyen

Learn to deploy an AKS cluster with OpenID Connect (OIDC) issuer and managed identity, creating a Key Vault and secret, and configuring the managed identity for token federation. This integration allows workloads on AKS to securely access Azure resources like KeyVault using Kubernetes service accounts and Microsoft Entra Workload ID.

10:00 AM - 10:45 AM

Building WebAssembly/WASI application (SpinApp) on AKS

Thang Chung

This presentation will briefly overview a new era of serverless with WebAssembly, orchestrating on Kubernetes (so-called SpinKube). Subsequently, the architecture of SpinKube, and how it stands on the existing OSSes. Finally, we propose a model that can leverage the existing CNCF ecosystem (like Dapr, or KEDA) to build kind of event-driven apps.

10:45 AM - 11:30 AM

AI Services on Azure, a look into the future

Phi Huynh

In this session, we will discuss on the latest insights of generative AI and latest developments on Azure Open AI services, Azure ML, Azure AI Studio and Copilots and how we can build innovative apps on Azure using these services.
